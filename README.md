# site-test — Portfolio photo statique

Ce dépôt contient un portfolio 100% statique (HTML/CSS/JS) prêt pour GitHub Pages.

## Mettre vos photos
1. Ajoutez vos images dans `assets/img/`.
2. Conservez les noms suivants : `photo-01.jpg` à `photo-12.jpg`.
3. Si un fichier manque, un placeholder élégant s'affiche automatiquement.

## Activer GitHub Pages
1. Ouvrez `Settings` > `Pages` dans votre dépôt GitHub.
2. Dans **Build and deployment**, choisissez **Deploy from a branch**.
3. Sélectionnez votre branche (ex: `main`) et le dossier `/ (root)`.
4. Enregistrez puis attendez la publication de l'URL.
